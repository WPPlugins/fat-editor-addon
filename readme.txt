== Description ==

#### FAT Editor AddOn

*AddOn for wordpress richtext editor.*

FAT Editor AddOn is a  WordPress plugin which allows you create highlight text, text with twitter share, video embed (youtube and vimeo).
With popup shortcode, you can easily create highlight text, twitter share or image with pinterest share and easily embed youtube, vimeo video


#### FAT Editor AddOn features

- Easy use: FAT Editor AddOn use shortcode popup GUI what help you easy create or edit hightlight text, twitter share or video embed

- Responsive Ready: It has a high functionality on wide range of screen sizes in any devices. It can adapt any screen sizes for achieving best viewing case.

- Unlimited color: You can set your own color scheme or choose from custom css for highlight text or twitter share text.



== Installation ==

#### Installing the plugin
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **FAT Editor AddOn** and click "*Install now*"
1. Alternatively, download the plugin and upload the contents of `fat-editor-addon.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin
